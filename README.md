# Sales Forecasting Project
Sales forecasting using machine learning (time-series analysis)

## Overview
This project predicts future sales using historical business data.

## Features
- Data cleaning and preprocessing
- Time-based feature engineering
- Sales trend visualization
- Machine learning forecasting using Linear Regression

## Results
- Visualized sales trends over time
- Forecasted future sales and compared with actual values

## Business Use Case
- Plan inventory efficiently
- Manage staffing during peak periods
- Reduce overstocking or shortages

## Tools Used
- Python
- Pandas
- Matplotlib
- Scikit-learn
